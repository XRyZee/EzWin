O que é um REPOSITORIO no GitHub


É um lugar onde você pode armazenar seu código, seus arquivos e o histórico
de revisão de cada arquivo.
Os repositórios podem ter vários colaboradores e podem ser públicos ou particulares.
Você pode criar 2 tipos de repositório, publico e privado.
Projetos públicos qualquer um pode ver, forkar e baixar o código fonte.
Projetos privados apenas os usuários que você liberar terão acesso ao código.

Clonar Repositorio


Ao clonar um repositório, você copia o repositório do GitHub.com
para o computador local ou para uma máquina virtual remota ao criar um codespace.
